# Desafios extras

Esta pasta organiza os arquivos, resultados e evidências associados aos desafios extras da atividade.

## Bônus documentados no relatório

- API REST;
- Lambda local;
- SQS;
- multiplicação por constante;
- dados por clínica;
- monitoramento;
- separação entre contexto público e contexto secreto.

## Observação

A chave secreta deve permanecer apenas no cliente. Os componentes de API, servidor, fila, Lambda e bucket manipulam somente contexto público, metadados e ciphertexts.

Os arquivos e evidências finais dos bônus devem ser adicionados nesta pasta antes da entrega definitiva.
